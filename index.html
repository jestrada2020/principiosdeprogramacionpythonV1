29. Contar Vocales en una Cadena

1 Punto

Define una función que cuente el número de vocales (a, e, i, o, u) en una cadena.

Sugerencia: Itera sobre la cadena y verifica si cada carácter es una vocal.

def contar_vocales(cadena):
            contador = 0
            vocales = "aeiouAEIOU"
            for char in cadena:
                if char [...] vocales:
                    contador [...] 1
            return contador

texto_analizar = "Murcielago"
num_vocales = contar_vocales(texto_analizar)
mensaje = f"La cadena \"{texto_analizar}\" tiene {num_vocales} vocales."
