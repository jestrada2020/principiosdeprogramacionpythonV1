Ejemplos Básicos para Definir Funciones

Aprende cómo crear aplicaciones web reactivas con Shiny for Python a través de este ejemplo práctico que demuestra funciones definidas por el usuario.

Ejemplo: Evaluador de Funciones Matemáticas
import numpy as np
import math
from shiny import App, render, reactive, ui

app_ui = ui.page_sidebar(
    ui.sidebar(
        ui.input_slider("n", "N", 0, 100, 20),
        ui.input_text("formula1", "Formula del usuario ", "3*x**2 - 3*x + 2"),
        ui.input_numeric("eval_x_f", "Evaluar f(x) en:", 0)),
    ui.output_text_verbatim("result_f"),
)

# Definir la función de servidor
def server(input, output, session):
    @output
    @render.code
    def txt():
        return f"n*2 is {input.n() * 2}"

    @output
    @render.text
    def result_f():
        x_value = input.eval_x_f()*input.n()
        result = eval(input.formula1(), {"x": x_value, "np": np, "math": math})
        return f"f({x_value}) = {result}"
        
app = App(app_ui, server)
Explicación del Código:
🎛️ Interfaz de Usuario (UI)
  • ui.page_sidebar(): Crea una estructura con barra lateral
  • ui.input_slider(): Control deslizante para el valor N (0-100)
  • ui.input_text(): Campo de texto para introducir fórmulas matemáticas
  • ui.input_numeric(): Campo numérico para el punto de evaluación
  • ui.output_text_verbatim(): Área de salida para mostrar resultados
⚙️ Lógica del Servidor
  • @output y @render.text: Decoradores que definen funciones reactivas
  • input.n(): Obtiene el valor del slider
  • input.formula1(): Recupera la fórmula introducida por el usuario
  • eval(): Evalúa la expresión matemática de forma segura
  • x_value: Calcula el punto donde evaluar la función
🔍 Características Destacadas
Reactividad: Los resultados se actualizan automáticamente cuando cambias los valores de entrada
Flexibilidad: Permite introducir cualquier fórmula matemática válida en Python
Integración: Usa numpy y math para cálculos avanzados
Ejemplo 2: Recorrido de Arreglo Interactivo
import numpy as np
import time
from shiny import App, render, reactive, ui

# Datos del arreglo
datos = [15, 23, 8, 42, 91, 37, 66, 14, 73, 29]

app_ui = ui.page_sidebar(
    ui.sidebar(
        ui.input_action_button("iniciar", "Iniciar Recorrido", class_="btn btn-primary mb-3"),
        ui.input_action_button("siguiente", "Siguiente Elemento", class_="btn btn-success mb-3"),
        ui.input_action_button("reiniciar", "Reiniciar", class_="btn btn-warning mb-3"),
        ui.input_slider("velocidad", "Velocidad (ms)", 500, 3000, 1000, step=100),
    ),
    ui.div(
        ui.h4("Recorrido del Arreglo", class_="mb-3"),
        ui.output_ui("mostrar_arreglo"),
        ui.output_text("elemento_actual"),
        ui.output_text("estadisticas"),
        class_="p-4"
    )
)

def server(input, output, session):
    valores = reactive.Value({'indice': -1, 'terminado': False, 'suma': 0})
    
    @reactive.Effect
    @reactive.event(input.iniciar)
    def iniciar_recorrido():
        valores.set({'indice': 0, 'terminado': False, 'suma': datos[0]})
    
    @reactive.Effect
    @reactive.event(input.siguiente)
    def siguiente_elemento():
        estado = valores.get()
        if not estado['terminado'] and estado['indice'] < len(datos) - 1:
            nuevo_indice = estado['indice'] + 1
            nueva_suma = estado['suma'] + datos[nuevo_indice]
            valores.set({
                'indice': nuevo_indice, 
                'terminado': nuevo_indice >= len(datos) - 1,
                'suma': nueva_suma
            })
    
    @reactive.Effect
    @reactive.event(input.reiniciar)
    def reiniciar():
        valores.set({'indice': -1, 'terminado': False, 'suma': 0})
    
    @output
    @render.ui
    def mostrar_arreglo():
        estado = valores.get()
        elementos = []
        
        for i, valor in enumerate(datos):
            if i < estado['indice']:
                # Elementos ya procesados (verde)
                elementos.append(ui.span(f"{valor}", class_="badge bg-success me-2 p-2"))
            elif i == estado['indice']:
                # Elemento actual (azul)
                elementos.append(ui.span(f"{valor}", class_="badge bg-primary me-2 p-2 fs-5"))
            else:
                # Elementos por procesar (gris)
                elementos.append(ui.span(f"{valor}", class_="badge bg-secondary me-2 p-2"))
        
        return ui.div(*elementos, class_="mb-3")
    
    @output
    @render.text
    def elemento_actual():
        estado = valores.get()
        if estado['indice'] >= 0 and not estado['terminado']:
            return f"Elemento actual: {datos[estado['indice']]} (Posición: {estado['indice']})"
        elif estado['terminado']:
            return f"¡Recorrido completado! Último elemento: {datos[estado['indice']]}"
        else:
            return "Presiona 'Iniciar Recorrido' para comenzar"
    
    @output
    @render.text
    def estadisticas():
        estado = valores.get()
        if estado['indice'] >= 0:
            elementos_procesados = estado['indice'] + 1
            promedio = estado['suma'] / elementos_procesados if elementos_procesados > 0 else 0
            return f"Procesados: {elementos_procesados}/10 | Suma: {estado['suma']} | Promedio: {promedio:.2f}"
        else:
            return "Estadísticas aparecerán durante el recorrido"

app = App(app_ui, server)
Explicación del Ejemplo de Recorrido:
🔄 Control de Recorrido
  • input_action_button(): Botones para controlar el recorrido
  • reactive.Value(): Estado reactivo que mantiene el índice actual
  • @reactive.event(): Responde a clics de botones específicos
  • input_slider(): Control de velocidad de recorrido
📊 Visualización Dinámica
  • render.ui(): Actualiza la interfaz dinámicamente
  • Badge coloreados: Verde (procesado), Azul (actual), Gris (pendiente)
  • Estadísticas en tiempo real: Suma, promedio, elementos procesados
  • Estado del recorrido: Índice actual y estado de finalización
🎯 Características del Ejemplo
Interactividad: Control manual del recorrido paso a paso
Visualización: Colores diferentes para cada estado del elemento
Estadísticas: Cálculos en tiempo real durante el recorrido
Ejemplo 3: Recorrido de Listas con Diferentes Tipos
from shiny import App, render, reactive, ui

# Lista mixta con diferentes tipos de datos
frutas = ["🍎 Manzana", "🍌 Plátano", "🍊 Naranja", "🍓 Fresa", "🥝 Kiwi"]
numeros = [10, 25, 30, 45, 50]
booleanos = [True, False, True, True, False]

app_ui = ui.page_sidebar(
    ui.sidebar(
        ui.input_select(
            "tipo_lista", 
            "Selecciona el tipo de lista:",
            choices={"frutas": "🍎 Frutas", "numeros": "🔢 Números", "booleanos": "✅ Booleanos"}
        ),
        ui.input_action_button("recorrer", "Recorrer Lista", class_="btn btn-info mb-3"),
        ui.input_action_button("siguiente_item", "Siguiente Item", class_="btn btn-success mb-3"),
        ui.input_action_button("reset", "Reiniciar", class_="btn btn-warning mb-3"),
    ),
    ui.div(
        ui.h4("Recorrido de Lista", class_="mb-3"),
        ui.output_ui("mostrar_lista"),
        ui.output_text("item_actual"),
        ui.output_text("info_lista"),
        class_="p-4"
    )
)

def server(input, output, session):
    estado = reactive.Value({'indice': -1, 'lista_actual': [], 'tipo': ''})
    
    def obtener_lista(tipo):
        if tipo == "frutas":
            return frutas
        elif tipo == "numeros":
            return numeros
        else:
            return booleanos
    
    @reactive.Effect
    @reactive.event(input.recorrer)
    def iniciar_recorrido():
        lista = obtener_lista(input.tipo_lista())
        estado.set({'indice': 0, 'lista_actual': lista, 'tipo': input.tipo_lista()})
    
    @reactive.Effect
    @reactive.event(input.siguiente_item)
    def siguiente_item():
        estado_actual = estado.get()
        if estado_actual['indice'] >= 0 and estado_actual['indice'] < len(estado_actual['lista_actual']) - 1:
            estado.set({
                'indice': estado_actual['indice'] + 1,
                'lista_actual': estado_actual['lista_actual'],
                'tipo': estado_actual['tipo']
            })
    
    @reactive.Effect
    @reactive.event(input.reset)
    def reiniciar():
        estado.set({'indice': -1, 'lista_actual': [], 'tipo': ''})
    
    @output
    @render.ui
    def mostrar_lista():
        estado_actual = estado.get()
        if not estado_actual['lista_actual']:
            return ui.p("Selecciona una lista y presiona 'Recorrer Lista'")
        
        items = []
        for i, item in enumerate(estado_actual['lista_actual']):
            if i < estado_actual['indice']:
                items.append(ui.span(f"{item}", class_="badge bg-success me-2 p-2 fs-6"))
            elif i == estado_actual['indice']:
                items.append(ui.span(f"{item}", class_="badge bg-primary me-2 p-2 fs-5"))
            else:
                items.append(ui.span(f"{item}", class_="badge bg-secondary me-2 p-2"))
        
        return ui.div(*items, class_="mb-3")
    
    @output
    @render.text
    def item_actual():
        estado_actual = estado.get()
        if estado_actual['indice'] >= 0 and estado_actual['lista_actual']:
            item = estado_actual['lista_actual'][estado_actual['indice']]
            return f"Item actual: {item} (Índice: {estado_actual['indice']}) | Tipo: {type(item).__name__}"
        return "Ningún item seleccionado"
    
    @output
    @render.text
    def info_lista():
        estado_actual = estado.get()
        if estado_actual['lista_actual']:
            total = len(estado_actual['lista_actual'])
            procesados = estado_actual['indice'] + 1 if estado_actual['indice'] >= 0 else 0
            return f"Lista: {estado_actual['tipo'].title()} | Total: {total} items | Procesados: {procesados}/{total}"
        return ""

app = App(app_ui, server)
Ejemplo 4: Definición de Matriz 3x2
import numpy as np
from shiny import App, render, reactive, ui

app_ui = ui.page_sidebar(
    ui.sidebar(
        ui.input_select(
            "tipo_matriz",
            "Tipo de Matriz:",
            choices={
                "manual": "📝 Manual",
                "zeros": "0️⃣ Ceros",
                "ones": "1️⃣ Unos",
                "random": "🎲 Aleatoria"
            }
        ),
        ui.h5("Valores Manuales (3x2):", class_="mt-3"),
        ui.input_numeric("m00", "Fila 0, Col 0:", value=1),
        ui.input_numeric("m01", "Fila 0, Col 1:", value=2),
        ui.input_numeric("m10", "Fila 1, Col 0:", value=3),
        ui.input_numeric("m11", "Fila 1, Col 1:", value=4),
        ui.input_numeric("m20", "Fila 2, Col 0:", value=5),
        ui.input_numeric("m21", "Fila 2, Col 1:", value=6),
        ui.input_action_button("crear_matriz", "Crear Matriz", class_="btn btn-primary mt-3"),
    ),
    ui.div(
        ui.h4("Matriz 3x2", class_="mb-3"),
        ui.output_ui("mostrar_matriz"),
        ui.output_text("info_matriz"),
        ui.output_text("propiedades_matriz"),
        class_="p-4"
    )
)

def server(input, output, session):
    matriz = reactive.Value(np.array([[1, 2], [3, 4], [5, 6]]))
    
    @reactive.Effect
    @reactive.event(input.crear_matriz)
    def crear_matriz():
        tipo = input.tipo_matriz()
        
        if tipo == "manual":
            nueva_matriz = np.array([
                [input.m00(), input.m01()],
                [input.m10(), input.m11()],
                [input.m20(), input.m21()]
            ])
        elif tipo == "zeros":
            nueva_matriz = np.zeros((3, 2))
        elif tipo == "ones":
            nueva_matriz = np.ones((3, 2))
        else:  # random
            nueva_matriz = np.random.randint(1, 20, size=(3, 2))
        
        matriz.set(nueva_matriz)
    
    @output
    @render.ui
    def mostrar_matriz():
        m = matriz.get()
        filas = []
        
        for i in range(3):
            columnas = []
            for j in range(2):
                valor = m[i, j]
                columnas.append(
                    ui.span(
                        f"{valor:.0f}" if valor == int(valor) else f"{valor:.2f}",
                        class_="badge bg-info me-2 p-3 fs-6",
                        style="min-width: 60px;"
                    )
                )
            filas.append(ui.div(*columnas, class_="mb-2"))
        
        return ui.div(*filas, class_="mb-3")
    
    @output
    @render.text
    def info_matriz():
        m = matriz.get()
        return f"Dimensiones: {m.shape} | Elementos totales: {m.size}"
    
    @output
    @render.text
    def propiedades_matriz():
        m = matriz.get()
        suma = np.sum(m)
        promedio = np.mean(m)
        maximo = np.max(m)
        minimo = np.min(m)
        return f"Suma: {suma:.2f} | Promedio: {promedio:.2f} | Máximo: {maximo:.2f} | Mínimo: {minimo:.2f}"

app = App(app_ui, server)
Ejemplo 5: Recorrido de Matriz 3x2 por Filas y Columnas
import numpy as np
from shiny import App, render, reactive, ui

# Matriz ejemplo 3x2
matriz_ejemplo = np.array([[10, 20], [30, 40], [50, 60]])

app_ui = ui.page_sidebar(
    ui.sidebar(
        ui.input_select(
            "modo_recorrido",
            "Modo de Recorrido:",
            choices={
                "filas": "📊 Por Filas",
                "columnas": "📈 Por Columnas"
            }
        ),
        ui.input_action_button("iniciar_recorrido", "Iniciar Recorrido", class_="btn btn-primary mb-3"),
        ui.input_action_button("siguiente_paso", "Siguiente Paso", class_="btn btn-success mb-3"),
        ui.input_action_button("reiniciar_todo", "Reiniciar", class_="btn btn-warning mb-3"),
        ui.h6("Estado del Recorrido:", class_="mt-3"),
        ui.output_text("posicion_actual"),
    ),
    ui.div(
        ui.h4("Recorrido de Matriz 3x2", class_="mb-3"),
        ui.output_ui("visualizar_matriz"),
        ui.output_text("elemento_actual"),
        ui.output_text("estadisticas_recorrido"),
        ui.output_ui("secuencia_recorrido"),
        class_="p-4"
    )
)

def server(input, output, session):
    estado_recorrido = reactive.Value({
        'fila': -1, 'columna': -1, 'completado': False, 
        'modo': 'filas', 'secuencia': []
    })
    
    def siguiente_posicion(fila, columna, modo):
        if modo == "filas":
            if columna < 1:
                return fila, columna + 1
            elif fila < 2:
                return fila + 1, 0
            else:
                return -1, -1  # Completado
        else:  # columnas
            if fila < 2:
                return fila + 1, columna
            elif columna < 1:
                return 0, columna + 1
            else:
                return -1, -1  # Completado
    
    @reactive.Effect
    @reactive.event(input.iniciar_recorrido)
    def iniciar():
        modo = input.modo_recorrido()
        estado_recorrido.set({
            'fila': 0, 'columna': 0, 'completado': False,
            'modo': modo, 'secuencia': [(0, 0)]
        })
    
    @reactive.Effect
    @reactive.event(input.siguiente_paso)
    def siguiente():
        estado = estado_recorrido.get()
        if not estado['completado']:
            nueva_fila, nueva_col = siguiente_posicion(
                estado['fila'], estado['columna'], estado['modo']
            )
            
            if nueva_fila == -1:  # Completado
                estado_recorrido.set({
                    **estado, 'completado': True
                })
            else:
                nueva_secuencia = estado['secuencia'] + [(nueva_fila, nueva_col)]
                estado_recorrido.set({
                    **estado, 'fila': nueva_fila, 'columna': nueva_col,
                    'secuencia': nueva_secuencia
                })
    
    @reactive.Effect
    @reactive.event(input.reiniciar_todo)
    def reiniciar():
        estado_recorrido.set({
            'fila': -1, 'columna': -1, 'completado': False,
            'modo': 'filas', 'secuencia': []
        })
    
    @output
    @render.ui
    def visualizar_matriz():
        estado = estado_recorrido.get()
        filas_ui = []
        
        for i in range(3):
            columnas_ui = []
            for j in range(2):
                valor = matriz_ejemplo[i, j]
                
                # Determinar el estilo según el estado
                if estado['fila'] == i and estado['columna'] == j and not estado['completado']:
                    clase = "badge bg-danger p-3 fs-5"  # Actual (rojo)
                elif (i, j) in estado['secuencia']:
                    clase = "badge bg-success p-3 fs-6"  # Visitado (verde)
                else:
                    clase = "badge bg-secondary p-3 fs-6"  # No visitado (gris)
                
                columnas_ui.append(
                    ui.span(
                        f"[{i},{j}] {valor}",
                        class_=clase,
                        style="min-width: 80px; margin: 2px;"
                    )
                )
            filas_ui.append(ui.div(*columnas_ui, class_="mb-2"))
        
        return ui.div(*filas_ui)
    
    @output
    @render.text
    def elemento_actual():
        estado = estado_recorrido.get()
        if estado['fila'] >= 0 and not estado['completado']:
            valor = matriz_ejemplo[estado['fila'], estado['columna']]
            return f"Elemento actual: [{estado['fila']},{estado['columna']}] = {valor}"
        elif estado['completado']:
            return "¡Recorrido completado!"
        return "Presiona 'Iniciar Recorrido' para comenzar"
    
    @output
    @render.text
    def posicion_actual():
        estado = estado_recorrido.get()
        if estado['fila'] >= 0:
            return f"Posición: Fila {estado['fila']}, Columna {estado['columna']}"
        return "Sin iniciar"
    
    @output
    @render.text
    def estadisticas_recorrido():
        estado = estado_recorrido.get()
        total_elementos = 6
        visitados = len(estado['secuencia'])
        return f"Modo: {estado['modo'].title()} | Visitados: {visitados}/{total_elementos}"
    
    @output
    @render.ui
    def secuencia_recorrido():
        estado = estado_recorrido.get()
        if estado['secuencia']:
            secuencia_items = []
            for i, (f, c) in enumerate(estado['secuencia']):
                valor = matriz_ejemplo[f, c]
                secuencia_items.append(
                    ui.span(f"[{f},{c}]→{valor}", class_="badge bg-info me-1 p-2 text-sm")
                )
            return ui.div(
                ui.h6("Secuencia de Recorrido:", class_="mb-2"),
                ui.div(*secuencia_items)
            )
        return ui.div()

app = App(app_ui, server)

Conceptos Clave de Shiny for Python

¿Qué es Shiny for Python?

Shiny for Python es un framework web que permite crear aplicaciones web interactivas usando únicamente Python. Originalmente desarrollado para R, Shiny ha sido adaptado para Python, permitiendo a los científicos de datos y desarrolladores crear dashboards y aplicaciones web reactivas sin necesidad de conocimientos profundos de HTML, CSS o JavaScript.

✨ Características Principales
  • • Reactividad automática
  • • Interfaz declarativa
  • • Sin JavaScript requerido
  • • Integración con Python científico
🎯 Casos de Uso
  • • Dashboards interactivos
  • • Herramientas de análisis
  • • Aplicaciones científicas
  • • Prototipos rápidos

Arquitectura de Shiny

UI (Interfaz)

Define la estructura visual y los elementos de entrada

Server (Servidor)

Contiene la lógica reactiva y procesamiento de datos

Reactividad

Actualización automática basada en cambios de entrada

def server(input, output, session):
    # Lista para guardar el historial
    historial_calculos = []
    
    @output
    @render.text  
    def operacion_actual():
        return f"{input.num1()} {obtener_simbolo()} {input.num2()}"
    
    @output
    @render.text
    def resultado_final():
        if input.calcular() > 0:  # Solo si se ha presionado el botón
            resultado = realizar_calculo()
            agregar_al_historial()
            return f"Resultado: {resultado}"
        return "Presiona 'Calcular' para ver el resultado"
    
    @output
    @render.text
    def historial():
        if historial_calculos:
            return f"Último cálculo: {historial_calculos[-1]}"
        return ""
    
    def obtener_simbolo():
        simbolos = {
            "suma": "+",
            "resta": "-", 
            "multiplicacion": "×",
            "division": "÷"
        }
        return simbolos.get(input.operacion(), "+")
    
    def realizar_calculo():
        num1, num2 = input.num1(), input.num2()
        operacion = input.operacion()
        
        if operacion == "suma":
            return num1 + num2
        elif operacion == "resta":
            return num1 - num2
        elif operacion == "multiplicacion":
            return num1 * num2
        elif operacion == "division":
            return round(num1 / num2, 2) if num2 != 0 else "Error: División por cero"
    
    def agregar_al_historial():
        resultado = realizar_calculo()
        calculo = f"{input.num1()} {obtener_simbolo()} {input.num2()} = {resultado}"
        historial_calculos.append(calculo)

app = App(app_ui, server)

Sistema de Reactividad

La reactividad es el corazón de Shiny. Permite que las salidas se actualicen automáticamente cuando cambian las entradas, sin necesidad de recargar la página.

🔄 Tipos de Elementos Reactivos
Reactive Sources

Elementos que generan eventos (inputs)

Reactive Conductors

Valores calculados que dependen de otros

Reactive Endpoints

Elementos que responden (outputs)

⚙️ Decoradores Principales
@render.text

Para salidas de texto

@render.plot

Para gráficos

@reactive.Calc

Para cálculos reactivos

@reactive.Effect

Para efectos secundarios

# Ejemplo de reactividad avanzada
from shiny import App, ui, render, reactive

def server(input, output, session):
    # Cálculo reactivo
    @reactive.Calc
    def datos_procesados():
        # Se recalcula automáticamente cuando input.n() cambia
        return [x * input.n() for x in range(1, 11)]
    
    # Salida que usa el cálculo reactivo
    @output
    @render.text
    def resultado():
        datos = datos_procesados()  # Usa el cálculo reactivo
        return f"Suma total: {sum(datos)}"

    ),
    
    # Contenido principal
    ui.h2("Dashboard de Ventas", class_="mb-4"),
    
    # Métricas principales
    ui.row(
        ui.column(3, ui.value_box("Total Ventas", ui.output_text("total_ventas"), theme="primary")),
        ui.column(3, ui.value_box("Promedio Diario", ui.output_text("promedio_diario"), theme="success")),
        ui.column(3, ui.value_box("Máximo", ui.output_text("maximo"), theme="warning")),
        ui.column(3, ui.value_box("Mínimo", ui.output_text("minimo"), theme="danger"))
    ),
    
    # Gráficos
    ui.row(
        ui.column(8, ui.output_plot("grafico_principal")),
        ui.column(4, ui.output_plot("grafico_categoria"))
    ),
    
    # Tabla de datos
    ui.h3("Datos Filtrados"),
    ui.output_table("tabla_datos")
)

def server(input, output, session):
    
    # Datos reactivos filtrados
    @reactive.Calc
    def datos_filtrados():
        df = data.copy()
        
        # Filtrar por fecha
        inicio, fin = input.fecha_rango()
        df = df[(df['fecha'] >= inicio) & (df['fecha'] <= fin)]
        
        # Filtrar por categoría
        if input.categorias():
            df = df[df['categoria'].isin(input.categorias())]
        
        # Filtrar por región
        if input.region() != "Todas":
            df = df[df['region'] == input.region()]
            
        return df
    
    # Métricas calculadas
    @output
    @render.text
    def total_ventas():
        df = datos_filtrados()
        return f"${df['ventas'].sum():,.0f}"
    
    @output
    @render.text
    def promedio_diario():
        df = datos_filtrados()
        return f"${df['ventas'].mean():,.0f}"
    
    @output
    @render.text
    def maximo():
        df = datos_filtrados()
        return f"${df['ventas'].max():,.0f}"
    
    @output
    @render.text
    def minimo():
        df = datos_filtrados()
        return f"${df['ventas'].min():,.0f}"
    
    # Gráfico principal
    @output
    @render.plot
    def grafico_principal():
        df = datos_filtrados()
        
        fig, ax = plt.subplots(figsize=(10, 6))
        
        if input.tipo_grafico() == "linea":
            ventas_diarias = df.groupby('fecha')['ventas'].sum().reset_index()
            ax.plot(ventas_diarias['fecha'], ventas_diarias['ventas'])
            ax.set_title("Evolución de Ventas por Día")
            
        elif input.tipo_grafico() == "barras":
            ventas_categoria = df.groupby('categoria')['ventas'].sum()
            ax.bar(ventas_categoria.index, ventas_categoria.values)
            ax.set_title("Ventas por Categoría")
            
        else:  # histograma
            ax.hist(df['ventas'], bins=30, alpha=0.7)
            ax.set_title("Distribución de Ventas")
        
        ax.tick_params(axis='x', rotation=45)
        plt.tight_layout()
        return fig
    
    # Gráfico por categoría
    @output
    @render.plot
    def grafico_categoria():
        df = datos_filtrados()
        
        fig, ax = plt.subplots(figsize=(6, 6))
        ventas_cat = df.groupby('categoria')['ventas'].sum()
        
        ax.pie(ventas_cat.values, labels=ventas_cat.index, autopct='%1.1f%%')
        ax.set_title("Distribución por Categoría")
        
        return fig
    
    # Tabla de datos
    @output
    @render.table
    def tabla_datos():
        df = datos_filtrados()
        return df.head(10).round(2)  # Mostrar solo primeras 10 filas

# Crear y ejecutar la aplicación
app = App(app_ui, server)

if __name__ == "__main__":
    app.run()

Reactividad Avanzada y Patrones

🔄 Valores Reactivos Personalizados
reactive.Value()

Crea valores reactivos personalizados que puedes modificar

# Crear valor reactivo personalizado
rv = reactive.Value(0)
rv.set(5)  # Establecer valor
current_val = rv.get()  # Obtener valor

# Uso en función reactiva
@reactive.Calc
def computed():
    return rv.get() * 2
reactive.isolate()

Rompe la reactividad para evitar dependencias no deseadas

@render.text
def output():
    # Reactivo: se ejecuta cuando input.x() cambia
    x = input.x()
    
    # No reactivo: no causa re-ejecución
    with reactive.isolate():
        y = input.y()
    
    return f"X: {x}, Y: {y}"
⚡ Eventos y Efectos Secundarios
@reactive.event

Controla exactamente cuándo se ejecuta una función reactiva

# Solo se ejecuta cuando se hace clic en el botón
@reactive.Calc
@reactive.event(input.update_btn)
def expensive_calculation():
    # Solo se recalcula al hacer clic
    return complex_operation()

# Múltiples eventos
@reactive.effect
@reactive.event(input.btn1, input.btn2)
def multi_trigger():
    print("Triggered by either button")
@reactive.Effect

Ejecuta efectos secundarios reactivamente (logs, actualizaciones, etc.)

@reactive.effect
def log_changes():
    # Se ejecuta cada vez que input.data() cambia
    data = input.data()
    print(f"Data changed to: {data}")
    # Guardar en base de datos, enviar notificaciones, etc.

# Efecto condicional
@reactive.effect
def conditional_effect():
    if input.enable_logging():
        print(f"Value: {input.value()}")

Diferencias con Shiny for R

Shiny for Python mantiene la misma filosofía que Shiny for R pero adapta la sintaxis a las convenciones de Python.

Shiny for R
Inputs input$x # Acceso como lista
Outputs output$y <- renderText(z())
Reactive Values reactiveVal(1)
Expressions z <- reactive({input$x + 1})
Events eventReactive(input$go, {...})
Shiny for Python
Inputs input.x() # Llamada de función
Outputs @render.text
def y(): return z()
Reactive Values reactive.Value(1)
Expressions @reactive.Calc
def z(): return input.x()+1
Events @reactive.event(input.go)
@reactive.Calc
def func(): ...
🔑 Diferencias Clave
  • • Decoradores: Python usa @decoradores para definir outputs en lugar de asignaciones
  • • Funciones: Los inputs se llaman como funciones: input.x() vs input$x
  • • Organización: Las funciones están organizadas en submódulos (ui.input_*, @render.*)
  • • Navegación: Se usa nav_*() en lugar de *Tab() para pestañas dinámicas

🌐 ¿Qué es Shinylive?

Shinylive utiliza WebAssembly para ejecutar Python completamente en el navegador, sin necesidad de un servidor Python. Es perfecto para demos, prototipos y educación.

✨ Ventajas
  • • Sin servidor Python requerido
  • • Carga instantánea
  • • Ideal para prototipos
  • • Fácil de compartir
  • • Integración con Quarto

Visita shinylive.io para editar y probar aplicaciones

2. Exportar Aplicación shinylive export myapp site
3. Desplegar

Sube a GitHub Pages, Netlify, o cualquier hosting estático

📝 Integración con Quarto

Puedes embeber aplicaciones Shinylive directamente en documentos Quarto, blogs y sitios web: 

```{shinylive-python}
#| standalone: true
#| components: [editor, viewer]

from shiny import App, render, ui
import matplotlib.pyplot as plt
import numpy as np

app_ui = ui.page_fluid(
    ui.input_slider("n", "Sample Size", 0, 1000, 20),
    ui.output_plot("dist")
)

def server(input, output, session):
    @render.plot
    def dist():
        x = np.random.randn(input.n())
        plt.hist(x, bins=30)
        plt.title("Random Distribution")

app = App(app_ui, server)
```
